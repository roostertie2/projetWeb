Pour démarrer l'application il n'y a pas de prérequis, il faut uniquement run l'application. Il y a aussi un front end que tu peux démarrer.

pour se connecter a h2: http://localhost:8080/h2-console

username: sa
il n'y a pas de password

pour postman: http://localhost:8080/

Puis après c'est le nom de la classe modèle avec un s et après le mapping, par exemple: http://localhost:8080/cars/create
